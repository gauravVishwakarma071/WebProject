CSS Overflow Property Example

Overflow: visible (default)

This is some long content that will not fit inside the box. It will overflow and still be visible outside of the box boundary. Extra text just to make it overflow and show the difference clearly.

Overflow: hidden

This is some long content that will not fit inside the box. The extra text is clipped and cannot be seen. You cannot scroll to read more.

Overflow: scroll

This is some long content that will not fit inside the box. Scrollbars will always appear, even if the content fits. Add more text so that it clearly overflows and requires scrolling.

Overflow: auto

This is some long content that will not fit inside the box. Scrollbars will appear only if needed. Add more text so that it clearly overflows and requires scrolling.
